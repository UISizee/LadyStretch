Power BI

Скачайте и установите Power BI Desktop с сайта Майкрософт.

Загрузите версию Power BI Desktop, соответствующую архитектуре (x86 или x64) вашей ОС Windows. Запустите EXE-установщик и выполните шаги установки.

Поддерживаемая операционная система:

Windows 10; Windows Server 2012 R2; Windows Server 2012; Windows 8; Windows 8.1; Windows Server 2016; Windows Server 2019; Windows 11.

Microsoft Power BI Desktop требует наличия браузера Internet Explorer версии 10 или более поздних версий. Microsoft Power BI Desktop доступен для 32-разрядных (x86) и 64-разрядных (x64) платформ.
